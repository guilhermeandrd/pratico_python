# pratico_python

-- para uso:

instale python e pygame zero

rode com: pgzrun main.py

-- classe player e enemy:

usada para ter mais controle sobre o objeto player, assim como a classe de inimigos, onde podemos usar a colisão com mais facilidade

-- classe platforme e hazard:

criadas usando rect e actor, para termos colisão e determinação de espaço e uso de imagem. Além disso, há essa separação pois hazard é uma área que causa o game over

-- mecânicas:

além disso tudo, o jogo tem a mecânica principal de ser um toque uma morte nos inimigos, além de que eles aumentam de velocidade conforme o tempo passa, então não adianta ficar pulando sempre, pois uma hora será game over.
